# dotfiles
Arch Linux下的一些配置文件。

# 常用的软件

网页浏览器：firefox-nightly、ungoogled-chromium

文本编辑器：vim、kate、typora

命令行解释器：zsh

数学工具：geogebra

媒体播放器：mpv

终端复用器：tmux

远程登录：ssh、mosh

图片处理：gimp

邮件：gmail

输入法：fcitx5

即时通讯：telegram-desktop-megumifox、electron-qq、freechat-git

音乐播放器：lx-music-desktop

BT客户端：qbittorrent-enhanced

内核：linux-cacule-rdb-llvm
